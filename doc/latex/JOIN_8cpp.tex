{\footnotesize\ttfamily int main (\begin{DoxyParamCaption}\item[{int}]{argc,  }\item[{char $\ast$}]{argv\mbox{[}$\,$\mbox{]} }\end{DoxyParamCaption})}

This program reads an undefined number of \mbox{\hyperlink{classLanguage}{Language}} objects from the files passed as parameters to \mbox{\hyperlink{JOIN_8cpp_a0ddf1224851353fc92bfbff6f499fa97}{main()}}. It obtains as result the union of all the input \mbox{\hyperlink{classLanguage}{Language}} objects. The result is then sorted by decreasing order of frequency and alphabetical order of bigrams when there is any tie in frequencies. Finally, the resulting \mbox{\hyperlink{classLanguage}{Language}} is saved in an output file. The program must have at least an output file. Running example\+: \begin{quote}
JOIN \mbox{[}-\/t$\vert$-\/b\mbox{]} \mbox{[}-\/o $<$output\+File.\+bgr$>$\mbox{]} $<$file1.\+bgr$>$ \mbox{[}$<$file2.\+bgr$>$ ... $<$filen.\+bgr$>$\mbox{]} \end{quote}
